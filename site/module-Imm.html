Methods

(inner) List(records, args) → {Imm.List}

Returns an Imm list. Keys are always sorted in alphabetical order. This means that original order of array given is not respected.
Parameters:
Name Type Description
records Array Array of records
args Object Optional arguments
Properties
Name Type Default Description
key String id Optional name of id key e.g. _id
Source:
  • List.js, line 14
Returns:
Imm.List
Type
Imm.List
Example
var records = [{id: 1, label: 'Sam'}, {...}];
var list = Imm.List(records);

// Imm assumes that the id key is called `id`. You can provide an optional argument:

var list = Imm.List(records, {key: '_id'});

Example

var record = Imm.Obj(data);

To get back a mutable JS object use `asMutable`:

var data = {id: 1, label: 'Sam'};
var immutableRecord = Imm.Obj(data);
var mutableRecord = immutableRecord.asMutable();
